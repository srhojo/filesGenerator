# FilesGenerator
Proyecto Spring para generar PDF, EXCEL y Txt.

## PDF
- **Description:** Se ha usado las librerias "Flying Saucer" y "Jade4J". A partir de una plantilla Jade generemos un PDF con un modelo de datos que le pasamos de ejemplo.
El modelo de datos debe ser un HasMap<String,Object> con el nombre del objeto que va a usar la plantilla jade y el objeto. Esté último puede ser cualquier Objeto java y se pueden acceder a sus funciones publicas (ej; UtilsLocalTime, UtilsMoney).

- **url:** http://localhost:8080/filesGenerator/files/pdf
